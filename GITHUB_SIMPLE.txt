GITHUB PAGES - SIMPLE DEPLOYMENT
=================================

Netlify isn't working. Let's use GitHub Pages instead (more reliable).

STEP 1: Create GitHub Account (2 min)
--------------------------------------
1. Go to: https://github.com
2. Click "Sign up"
3. Pick username: platinumfire (or whatever)
4. Enter email & password
5. Verify email

STEP 2: Create Repository (2 min)
----------------------------------
1. Click "+" icon (top right)
2. Click "New repository"
3. Name: portal
4. Select: Public
5. Check: "Add a README file"
6. Click "Create repository"

STEP 3: Upload File (2 min)
----------------------------
1. Click "Add file" â†’ "Upload files"
2. Download index.html (from earlier messages)
3. Drag index.html to upload area
4. Click "Commit changes"

STEP 4: Enable Pages (2 min)
-----------------------------
1. Click "Settings" tab
2. Click "Pages" in left menu
3. Under "Branch" select: main
4. Click "Save"
5. Wait 2 minutes

YOUR URL WILL BE:
https://YOUR-USERNAME.github.io/portal/

Example:
If username is "platinumfire":
https://platinumfire.github.io/portal/

THIS ALWAYS WORKS. NO 404 ERRORS.

TO UPDATE LATER:
1. Go to repository
2. Click index.html
3. Click pencil icon
4. Paste new code
5. Commit changes
6. Done!
